Komplexe Zahlen in der Gaußschen Zahlenebene

{% endblock %} {% block DEF1_HEAD %} Was sind komplexe Zahlen? {% endblock %} {% block DEF1 %}

Eine Folge $\left ( a_n \right )_{n=0}^\infty: \mathbb{N} \to \mathbb{R}$ ist eine Zuordnung einer natürlichen Zahl $n \in \mathbb{N}$ auf eine reelle Zahl $a_n \in \mathbb{R}$ vermöge $n \mapsto a_n$.

Die Zahl $a_n$ ist ein Folgenglied, also ein Element der Folge.

{% endblock %} {% block DEF2_HEAD %} Gaußsche Zahlenebene {% endblock %} {% block DEF2%}

Eine Folge $\left ( a_n \right )_{n=0}^\infty$ heißt monoton wachsend, wenn $a_{n+1} \geq a_n$ für alle $n \in \mathbb{N}$ gilt. Sie heißt streng monoton wachsend, wenn sogar $a_{n+1} > a_n$ für alle $n \in \mathbb{N}$ gilt.

Analog heißt eine Folge $\left ( a_n \right )_{n=0}^\infty$ monoton fallend, wenn $a_{n+1} \leq a_n$ für alle $n \in \mathbb{N}$ gilt und streng monoton fallend, wenn $a_{n+1} < a_n$ für alle $n \in \mathbb{N}$ gilt.

{% endblock %} {% block DEF3_HEAD %} Polarkoordinaten {% endblock %} {% block DEF3 %}

Eine Folge $\left ( a_n \right )_{n=0}^\infty$ ist nach oben beschränkt, wenn es eine obere Schranke $s \in \mathbb{R}$ gibt, sodass $a_n \leq s$ für alle $n \in \mathbb{N}$ gilt.

Eine Folge $\left ( a_n \right )_{n=0}^\infty$ ist nach unten beschränkt, wenn es eine untere Schranke $u \in \mathbb{R}$ gibt, sodass $a_n \geq u$ für alle $n \in \mathbb{N}$ gilt.

Eine Folge $\left ( a_n \right )_{n=0}^\infty$ ist beschränkt, wenn es eine obere und eine untere Schranke gibt.
