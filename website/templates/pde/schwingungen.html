Schwingungen einer Saite

{% endblock %} {% block DEF1_HEAD %} Die Schwingungsgleichung {% endblock %} {% block DEF1 %}

Die Schwingungsgleichung beschreibt, dass die Kraft, die an einer Stelle $x$ und zu einem Zeitpunkt $t$ auf die Saite wirkt, durch die rücktreibende Kraft aus der Verzerrung der Saite wirkt.

Aus dem zweiten Newtonschen Gesetz ergibt sich \begin{align*} \varrho u_{,tt} = P u_{,xx}, \end{align*} wobei $varrho$ die Längendichte der Saite angibt, $P$ die Vorspannung der Saite ist und $u(t,x)$ die Auslenkung der Saite an einem Ort $x$ zur Zeit $t$ angibt.

Auf der linken Seite der Gleichung finden wir das 2. Newtonsche Gesetz, auf der rechten die rücktreibende Kraft durch die Verzerrung.

{% endblock %} {% block DEF2_HEAD %} Randbedingungen {% endblock %} {% block DEF2%}

Bei einer Gitarrensaite sind die beiden Enden eingespannt und die Saite lässt sich dort nicht auslenken.

Eine solche Einspannung lässt sich durch Dirichlet-Randbedingungen beschreiben. Mit der zusätzlichen Festlegung, die Saite auf der Höhe 0 einzuspannen, erhalten wir die homogenen Dirichlet-Randbedingungen \begin{align*} u(t,a) = u(t,b)=0, \end{align*} wobei $x=a$ und $x=b$ die beiden Enden der Saite angeben.

{% endblock %} {% block DEF3_HEAD %} Anfangsbedingungen {% endblock %} {% block DEF3 %}

Die Schwingungsgleichung benötigt zwei Anfangsbedingungen: eine für die Auslenkung zu Beginn und eine für die Geschwindigkeit der Saite.

Mit den Anfangsbedingungen \begin{align*} u(0,x)&= u_0(x) , x \in \Omega \\ u_{,t}(0,x)&= v_0(x), x \in \Omega \end{align*} bekommen wir ein vollständiges Anfangsrandwertproblem.
