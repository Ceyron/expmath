Wellenausbreitung ohne Randbedingungen

{% endblock %} {% block DEF1_HEAD %} Wellengleichung {% endblock %} {% block DEF1 %}

Wir betrachten eine Wellengleichung $$u_{,tt} = c^2 u_{,xx} \quad \text{für} x \in \mathbb{R}, t>0.$$

Zusammen mit den Anfangsbedingungen $u(0,x)=u_0(x)$ und $u_{,t}(0,x)= u_1(x)$ erhalten wir ein vollständiges Anfangswertproblem.

{% endblock %} {% block DEF2_HEAD %} Lösungsansatz {% endblock %} {% block DEF2%}

Die Lösung $u(t,x)$ setzt sich aus einer nach rechtslaufenden Welle $F(x-ct)$ und einer nach linkslaufenden Welle $G(x+ct)$ zusammen, also $$u(t,x)= F(x-ct)+ G(x+ct).$$

Die Gestalt von $F$ und $G$ hängt von den Anfangsbedingungen $u_0(x)$ und $u_1(x)$ ab. Genauer gilt $$ u(t,x) = \\frac{1}{2} \left ( u_0(x-ct) + u_0(x+ct) + \frac{1}{c} \int_{x-ct}^{x+ct} u_1(y) \mathrm{d}y.$$
