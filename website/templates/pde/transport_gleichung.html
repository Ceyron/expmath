Transportgleichung und Charakteristiken

{% endblock %} {% block DEF1_HEAD %} Transportgleichung {% endblock %} {% block DEF1 %}

Eine Transportgleichung hat die Gestalt $u_{,t} + \\frac{\mathrm{d}}{\mathrm{d}x} (F(x,u)) =0$.

Die Funktion $F$ ist die Flussfunktion. Sie ist immer von $u$ und manchmal von $x$ abhängig.

Die Transportgleichung ist eine partielle Differentialgleichung 1. Ordnung.

{% endblock %} {% block DEF2_HEAD %} Methode der Charakteristiken {% endblock %} {% block DEF2%}

Ein Lösungsansatz zur Lösung einer Transportgleichung ist die Methode der Charakteristiken. Hierzu werden Charakteristiken $\xi(t)$ genau so definiert, dass anstelle partiellen Differentialgleichung zwei gewöhnliche Differentialgleichungen gelöst werden.

Genauer sind die Charakteristiken über die Gleichungen $$ \xi^\prime (t) = \frac{\partial}{\partial u} F(\xi(t), u(t,\xi(t))) $$ und $$ \\frac{\mathrm{d}}{\mathrm{d}t} u(t, xi(t))= - \\frac{\partial}{\partial x}F(\xi(t), u(t,\xi(t))) $$ definiert.

{% endblock %} {% block DEF3_HEAD %} Eigenschaften der Flussfunktion und der Lösung {% endblock %} {% block DEF3 %}

Durch die Definition der Charakteristiken ergibt sich, dass die Flussfunktion $F(\xi(t), u(t,\xi(t)))$ entland der Charakteristiken konstant ist.

Es gilt also $\\frac{\mathrm{d}{\mathrm{d}t} F(\xi(t), u(t,\xi(t)))=0.$

Im Fall $F= F(u)$ gilt ferner, dass die Lösung entlang der Charakteristiken konstant ist, also $\\frac{\mathrm{d}}{\mathrm{d}t} u(t, xi(t))=0$.
