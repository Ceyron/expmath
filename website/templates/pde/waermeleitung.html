{% endblock %} {% block DEF1_HEAD %} Wärmeleitung durch Diffusion {% endblock %} {% block DEF1 %}

Wärme fließt vom heißen Potential zum kalten. Das macht den Fluss durche ein Medium proportional zum negativen Gradienten des Temperaturfeldes $F \prop - \grad u$. Dieser Potentialausgleich heißt Diffusion.

{% endblock %} {% block DEF2_HEAD %} Vom Temperaturgradienten zum Fluss {% endblock %} {% block DEF2%} Zur quantitativen Beschreibung der Stärke des Flusses wird die konstitutive Gleichung benötigt. Diese entspricht der Proportionalitätskonstanten und enthält die Informationen zu potentieller Inhomogenität oder Anisotropie $ F = \mathbf{A} u$.

{% endblock %} {% block DEF3_HEAD %} Die Wärmeleitungsgleichung {% endblock %} {% block DEF3 %} Fließt Wärme, so ändert sich auch die Temperatur. Dieser Zusammenhang wird in der Kontinuitätsgleichung deutlich. Eine zeitliche Änderung der Temperatur wird durch eine Divergenz des Flussfeldes hervorgerufen $ u_t + \grad \dot [\mathbf{A} \grad u] = 0$.
