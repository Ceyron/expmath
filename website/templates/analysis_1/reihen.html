Partialsummen und Reihen

{% endblock %} {% block DEF1_HEAD %} Definition einer Reihe als Folge von Partialsummen {% endblock %} {% block DEF1 %}

Die Reihe oder unendliche Summe

$$\sum_{k = 0}^{\infty} a_k = a_0 + a_1 + a_2 + a_3 + \ldots $$

mit den Summanden $a_k$, $k\in \mathbb{N}$, ist die Folge $(s_n)_{n = 0}^{\infty}$ der Partialsummen

$$s_n = \sum_{k=0}^n a_k = a_0 + a_1 + a_2 + \ldots + a_n.$$

{% endblock %} {% block DEF2_HEAD %} Wann konvergiert eine Reihe? {% endblock %} {% block DEF2%}

Konvergiert die Partialsummenfolge $(s_n)_{n=0}^\infty$, so heißt die zugehörige Reihe konvergent.

Konvergiert die Reihe $\sum_{k = 0}^{\infty} a_k$, so bilden ihre Summanden $a_k$ eine Nullfolge.

Eine Reihe $\sum_{k = 0}^{\infty} a_k$ konvergiert absolut, wenn die Reihe $\sum_{k = 0}^{\infty} \lvert a_k\rvert$ der Absolutbeträge konvergiert.

{% endblock %} {% block DEF3_HEAD %} Konvergenzkriterien {% endblock %} {% block DEF3 %}

Um die Konvergenz von Reihen zu überprüfen, gibt es verschiedene Kriterien.

  • Vergleichskriterium
    • $\displaystyle\sum_{k = 0}^{\infty} b_k$ konvergente Majorante mit $\lvert a_k\rvert \leq \lvert b_k\rvert$ $\Rightarrow$ $\displaystyle\sum_{k = 0}^{\infty} a_k$ absolut konvergent
    • $\displaystyle\sum_{k = 0}^{\infty} b_k$ divergente Minorante mit $a_k \geq b_k$ $\Rightarrow$ $\displaystyle\sum_{k = 0}^{\infty} a_k$ divergent
  • Quotientenkriterium: $\limsup_{k \to \infty} \lvert\frac{a_{k+1}}{a_k}\rvert \begin{cases} < 1 & {\scriptstyle\text{ Reihe absolut konvergent}} \\ = 1 & {\scriptstyle\text{ keine Aussage möglich}} \\ > 1 & {\scriptstyle\text{ keine Aussage möglich}} \end{cases}$
  • Wurzelkriterium: $\limsup_{k\to\infty} \sqrt[k]{\lvert a_k \rvert} \begin{cases} < 1 & {\scriptstyle\text{ Reihe absolut konvergent}} \\ = 1 & {\scriptstyle\text{ keine Aussage möglich}} \\ > 1 & {\scriptstyle\text{ Reihe divergent}} \end{cases}$
  • Leibniz-Kriterium: wenn für eine Reihe $\displaystyle\sum_{k = 0}^{\infty} a_k$ die Bedingungen
    • $\lim_{k \to \infty} a_k = 0$
    • $(\lvert a_k\rvert)_{k=0}^\infty$ monoton fallend
    • alternierend
    $\displaystyle erfüllt sind, ist sie konvergent.
{% endblock %} {% block DEF4_HEAD %} Grenzwert {% endblock %} {% block DEF4 %}

Eine Folge $\left ( a_n \right )_{n=0}^\infty$ hat einen Grenzwert $a \in \mathbb{R}$, wenn $$ \forall \varepsilon >0 \ \exists N \, : \, \lvert a_n - a \rvert < \varepsilon \ \forall n \geq N$$ gilt.

Wenn ein Grenzwert für eine Folge existiert, schreiben wir $\lim_{n \to infty } a_n =a$ oder kurz $(a_n) \rightarrow a$. Die Folge konvergiert gegen den Grenzwert $a$.

Wenn dieser Grenzwert $a=0$ ist, so heißt die Folge Nullfolge.

Eine Folge, die nicht konvergiert, heißt divergente Folge.
