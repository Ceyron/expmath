Differenzierbarkeit von Funktionen

{% endblock %} {% block DEF1_HEAD %} Was ist Differenzierbarkeit? {% endblock %} {% block DEF1 %}

Eine Funktion $f: [a,b] \to \mathbb{R}$ heißt differenzierbar an der Stelle $x_0 \in (a,b)$, wenn der Differentialquotient $$ \lim_{h\to 0} \frac{f(x_0 + h) - f(x_0)}{h} = f'(x_0) = \frac{d f(x_0)}{d x}$$ existiert. Der Grenzwert bezeichnet die Steigung der Tangenten an den Graphen von $f$ an der Stelle $x_0$.

Der Grenzwert $f'(x_0)$ ist die Ableitung von $f$ an der Stelle $x_0$.

Die Funktion $f: [a,b] \to \mathbb{R}$ heißt stetig differenzierbar, wenn ihre Ableitung $f': [a,b] \to \mathbb{R}$ eine stetige Funktion ist. Die Menge aller stetig differenzierbaren Funktionen $f: [a,b] \to \mathbb{R}$ bilden den Vektorraum $C^1([a,b])$.

{% endblock %} {% block DEF2_HEAD %} Ableitungsregeln {% endblock %} {% block DEF2 %}

Aus dem Differentialquotienten lassen sich einige Ableitungsregeln herleiten, mit denen man einfacher rechnen kann.

Für zwei differenzierbare Funktionen $u,v: [a,b] \to \mathbb{R}$ gelten die folgenden Regeln:

  • Produktregel: $\displaystyle (uv)'(x) = u'v + uv'$
  • Kettenregel: $\displaystyle \frac{d}{d x} \ u(v(x)) = u'(v(x)) \cdot v'(x) = $ äußere $\cdot$ innere Ableitung
  • Quotientenregel: $\displaystyle \left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2} $
