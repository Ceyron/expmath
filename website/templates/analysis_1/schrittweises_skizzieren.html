Schrittweises Skizzieren von Funktionen

{% endblock %} {% block DEF1_HEAD %} Skizzieren von Funktionen {% endblock %} {% block DEF1 %}

Über viele Funktionen wissen wir bereits eine Menge. Welche Eigenschaften haben sie? Wie sieht die Funktion aus? Welche charakteristischen Punkte müssen wir berücksichtigen?

Allerdings sieht man nicht immer sofort vor dem inneren Auge, wie eine Funktion aussieht. Dies kann vor allem bei zusammengesetzten Funktionen schwierig sein.

{% endblock %} {% block DEF2_HEAD %} Zerlegen in Teilfunktionen {% endblock %} {% block DEF2%}

Bei Funktionen, die aus mehreren Teilen zusammengesetzt sind, kann man sich durch eine Rückführung auf bekannte Terme und Graphen Schritt für Schritt dem Funktionsgraphen nähern.

Dazu untersuchen wir die Funktion und suchen nach einem Term, dessen Graph wir kennen.

Durch schrittweise Erweiterung dieser Teilfunktion um die in der jeweiligen Funktion vorkommenden Terme entwickeln wir Teilgraphen, die mit jedem Schritt erweitert werden.

Wenn wir dies so lange wiederholen, bis die ursprüngliche Funktion betrachtet wird, landen wir schließlich beim gesuchten Funktionsgraphen.

{% endblock %} {% block BOKEH %} {{ script|safe }} {% endblock %} {% block QUEST1_HEAD %} Teil 1 - Erkundung der Grafik {% endblock %} {% block QUEST1 %}
  1. Schauen Sie sich die verschiedenen Funktionen an. Haben Sie ohne Nutzung der Grafik eine Vorstellung von den zugehörigen Funktionsgraphen?
  2. Gehen Sie die Schritte für die erste Funktion $\displaystyle\frac{1}{\sqrt{1+x^2}}$ durch. Welche Graphen erkennen Sie in den einzelnen Schritten?
  3. Fertigen Sie eine eigene Skizze an, indem Sie die Schritte aus der Grafik übertragen.
  4. Skizzieren Sie die Funktion $\displaystyle \sin^2\left(x+\frac{\pi}{2}\right)$.
  5. Notieren Sie dazu zunächst die jeweils vorkommenden Teilfunktionen.
  6. Skizzieren Sie die Teilfunktionen zu einer Funktion nacheinander in einem Koordinatensytem.
  7. Überprüfen Sie Ihre Skizzen mithilfe der Grafik.
  8. Fertigen Sie Skizzen der übrigen in der Grafik auftauchenden Funktionen $\displaystyle\frac{1}{x^2 - 2}$ und $\displaystyle\frac{{\rm{e}}^x}{1 + {\rm{e}}^x}$ an.
  9. Nutzen Sie auch hier Teilfunktionen und überprüfen Sie Ihre Skizzen mithilfe der Grafik.
{% endblock %} {% block ANS1 %} Antwort 1 {% endblock %} {% block QUEST2_HEAD %} Teil 2 - Zerlegen in Teilfunktionen {% endblock %} {% block QUEST2 %}
  1. Skizzieren Sie die Funktion $\displaystyle f(x) = \frac{1}{(x-3)^2}$.
    • Identifizieren Sie dazu zunächst die Teilfunktionen in $f$.
    • Skizzieren Sie die Funktionen $f_1(x) = x$, $f_2(x)=x-3$, $f_3(x)=(x-3)^2$ und schließlich $f_4(x) = f(x)$ nacheinander in einem Koordinatensystem.
  2. Skizzieren Sie die Funktion $\displaystyle g(x) = \frac{1}{(x+3)^2}$.
    • Identifizieren Sie dazu die Teilfunktionen $g$ und skizzieren Sie diese nacheinander in einem Koordinatensystem.
  3. Skizzieren Sie die Funktion $\displaystyle h(x) = \frac{1}{(x-3)^2} + \frac{1}{(x+3)^2}$. Nutzen Sie dazu die Skizzen von $f$ und $g$.
  4. Trainieren Sie das schrittweise Skizzieren, indem Sie sich weitere Beispiele ausdenken und dort jeweils Teilfunktionen identifizieren und skizzieren.
