Stetigkeit von Funktionen

{% endblock %} {% block DEF1_HEAD %} Was ist Stetigkeit? {% endblock %} {% block DEF1 %}

{% endblock %} {% block DEF2_HEAD %} Epsilon-Delta-Kriterium {% endblock %} {% block DEF2%}

Ist $f: D \to\mathbb{R}$ stetig im Punkt $z \in D$, so gilt $$\forall \varepsilon > 0 \ \exists \delta > 0 \ \forall x \in D: \ \lvert x - z \rvert < \delta \ \Rightarrow \ \lvert f(x) - f(z) \rvert < \varepsilon. $$ Das Intervall $(z-\delta, z+\delta)$ wird also in den $\varepsilon$-Schlauch um $f(z)$ abgebildet.

Eine Funktion $f:[a,b] \to \mathbb{R}$ oder $f: \mathbb{R} \to \mathbb{R}$ heißt stetig, wenn sie in jedem Punkt stetig ist.

{% endblock %} {% block DEF3_HEAD %} Grenzwerte von Funktionen {% endblock %} {% block DEF3 %}

$f: D \to \mathbb{R}$ und $(a,b)\subseteq D, z \in [a,b]$

Grenzwert von $f$ im Punkt $z$: Für jede Folge $(x_n)_{n=0}^\infty$ mit $x_n \in D \ \forall n \in \mathbb{N}$ und $\displaystyle\lim_{n \to \infty} x_n = z$ gilt $$\lim_{n \to \infty} f(x_n) = c$$ Notation: $\displaystyle \lim_{x \to z} f(x) = c$

Linksseitiger Grenzwert: $$\displaystyle\lim_{x \to z^-} f(x) = \lim_{n\to\infty} f(x_n) \ \ \forall (x_n)_{n=0}^\infty \to z \ , \ x_n < z$$

Rechtsseitiger Grenzwert: $$\displaystyle\lim_{x \to z^+} f(x) = \lim_{n\to\infty} f(x_n) \ \ \forall (x_n)_{n=0}^\infty \to z \ , \ x_n > z$$

Folgenkriterium: Die Funktion $f: D \to \mathbb{R}$ heißt stetig im Punkt $z \in D$, wenn für alle Folgen $(x_n)_{n=0}^\infty$, die gegen $z$ konvergieren, die Folge $(f(x_n))_{n=0}^\infty$ der zugehörigen Funktionswerte gegen $f(z)$ konvergiert. Stetigkeit bedeutet also eine Vertauschbarkeit von Funktionswert- und Grenzwertbildung an der Stelle $z$: $$\lim_{n\to\infty} f(x_n) = f(z) = f(\lim_{n\to\infty} x_n) \ \ \forall (x_n)_{n= 0}^\infty \to z.$$

{% endblock %} {% block DEF4_HEAD %} Arten von Unstetigkeitsstellen und Eigenschaften stetiger Funktionen {% endblock %} {% block DEF4 %}

Sprungstelle $$\lim_{x\to z^-} f(x) \neq \lim_{x \to z^+} f(x)\ , \ \text{beide Grenzwerte existieren}$$

Hebbare Unstetigkeitsstelle $$c = \lim_{x \to z^-} f(x) = \lim_{x \to z^+} f(x) \neq f(z)$$ behebbar durch Umdefinition $\tilde{f}(z) = c$

Hebbare Definitionslücke $$c = \lim_{x \to z^-} f(x) = \lim_{x \to z^+} f(x) \ , \ \text{aber hier: $f(z)$ nicht definiert}$$

Polstelle $$\lim_{x \to z^-} f(x) = \pm \infty \ \text{ und/oder } \ \lim_{x \to z^+} f(x) = \pm\infty $$

Zwischenwertsatz: Eine stetige Funktion $f: [a,b] \to \mathbb{R}$ nimmt jeden Wert zwischen $f(a)$ und $f(b)$ an.

Satz von Weierstraß: Sei $f:[a,b] \to \mathbb{R}$ stetig. Dann existieren Stellen $x_{\rm{min}},x_{\rm{max}}\in [a,b]$ mit $$f(x_{\rm{min}}) \leq f(x) \leq f(x_{\rm{max}}) \ \ \forall x \in [a,b].$$ Das bedeutet, dass stetige Funktionen auf abgeschlossenen Intervallen ihre Extrema annehmen. Sie sind beschränkt. Dies gilt nicht auf offenen oder halboffenen Intervallen.
