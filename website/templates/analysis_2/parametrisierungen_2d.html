Parametrisierung von Kurven in der Ebene

{% endblock %} {% block DEF1_HEAD %} Was ist eine Parametrisierung? {% endblock %} {% block DEF1 %}

Durch eine Parametrisierung $\gamma: I \subset \mathbb{R}^1 \rightarrow \mathbb{R}^2$ wird eine Kurve im zweidimensionalen Raum $\mathbb{R}^2$ in Abhängigkeit von einer Variablen $\varphi$ dargestellt. Zu jedem Parameterwert $t \in I$ gehört ein Punkt $\gamma (t) \in \mathbb{R}^2$.

Der Parameter $t$ nimmt alle Werte aus dem Intervall $I \subset \mathbb{R}^1$ an.

{% endblock %} {% block DEF2_HEAD %} Umgang mit der Skizze {% endblock %} {% block DEF2 %}

Wir betrachten Parametrisierungen der Form $t \mapsto \gamma(t) = \begin{pmatrix} \gamma_1(t) \\ \gamma_2(t) \end{pmatrix} \in \mathbb{R}^2$.

Die Art der Parametrisierung für die erste Komponente, also $\gamma_1(t)$ wird durch die obere Zeile ausgewählt.

Analog wird die Parametrisierung für die zweite Komponente $\gamma_2$ in der zweiten Zeile ausgewählt.

Der Bereich der Parametrisierung wird mit dem unteren Schieberegler verändert.

Optional können die Werte für die Koeffizienten $a,b,c,d \in \mathbb{R}$ nach Betätigung des Schalters "Parameter-Slider hinzuschalten" gewählt werden.
